net  user  

( မိမိစက်ထဲမှာ  Users  ဘယ်လောက်ရှိလဲဆိုတာကြည့် )

net  user  "username"
eg. net  user "Mg Mg"
  မောင်မောင် အကောင့် နဲ့ပတ်သက်တာအသေးစိတ်ကြည့်

net  user  /add  "username"  password
eg. net  user  /add "Aung Aung"   123

 အကောင့်သစ်တစ်ခုကို  Password  နဲ့တွဲဖွင့်

net   user  "Aung Aung"   *

 အောင်အောင်အကောင့်ကို  Password (Modify)  လုပ်  
{ cmd  တွင် ရေးသော  password  များအား မပေါ်) 

net  user  /delete  "username"
